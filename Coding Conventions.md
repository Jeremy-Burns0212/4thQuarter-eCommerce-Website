# Coding Conventions
## Acceptable Syntax for CSHTML
- functions, methods, classes, every nesting nestable type with curly braces, there must be an empty line between the closing brace and the next opening brace on the same level. like this:
```cshtml
<div class="text-center">
    <h1 class="fs-1 fw-normal">Welcome</h1>
</div>

<div class="alert alert-info" role="alert">
    This website is currently under construction
</div>
```

## Acceptable Syntax for CSS
```css
example {
    color = lightblue;
}
```
